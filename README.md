# 🐇 Rabbit Hole

> **The deeper you scroll, the stranger it gets.**

Rabbit Hole is an interactive React web application that lets users endlessly explore conspiracy theories by fetching information from Wikipedia. Instead of endlessly scrolling social media, users can dive into an infinite chain of fascinating historical mysteries, secret projects, unexplained events, and controversial topics while tracking how deep they've gone.

> **Educational Disclaimer:** This website presents conspiracy theories as documented Wikipedia topics for educational and entertainment purposes. Inclusion of a topic does **not** imply that its claims are true or verified. Users are encouraged to think critically and consult multiple reliable sources.

---

## ✨ Features

* 🔍 Search any conspiracy theory or mystery
* ♾️ Endless scrolling experience
* 📚 Fetches live article summaries from Wikipedia
* 🖼 Displays article images when available
* 📏 Depth meter showing how deep you've gone
* ⚡ Fast React interface
* 📱 Responsive design
* 🎨 Minimal, immersive UI

## 🚀 How It Works

1. User enters a conspiracy theory or mystery.
2. The application requests data from Wikipedia.
3. The article summary, image, and information are displayed.
4. Users continue scrolling or selecting related topics.
5. Each new article increases the Rabbit Hole depth.
6. The experience continues indefinitely.

The goal is to simulate the feeling of falling down an endless Wikipedia rabbit hole.

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/rabbit-hole.git
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

---

## 🌐 Wikipedia API

Rabbit Hole uses the public Wikipedia REST API to retrieve article summaries, descriptions, and images in real time. The application does not host or modify Wikipedia content. It simply displays publicly available information provided by Wikipedia.
